# Technical Details

## HTML Version

HTML5

## Architecture

The project follows a single-page portfolio design.

Users can navigate between sections using internal navigation links.

## Semantic HTML

The following semantic elements are used:

- header
- nav
- main
- section
- article
- footer

## Navigation System

Navigation links use anchor tags.

Example:

<a href="#about">About</a>

This allows users to jump directly to specific sections of the page.

## Images

Images are stored inside the images folder.

Example:

<img src="images/profile.jpg" alt="Profile Picture">

The alt attribute improves accessibility.

## Form Implementation

The contact form includes:

- Name field
- Email field
- Phone field
- Message field

HTML validation is implemented using:

required

attribute.

## Accessibility Features

- Semantic HTML
- Labels for form fields
- Alternative text for images
- Structured headings

## Browser Compatibility

Tested on:

- Google Chrome
- Microsoft Edge

The website works without requiring any additional libraries or frameworks.
