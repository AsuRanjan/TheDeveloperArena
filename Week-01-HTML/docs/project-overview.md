# Project Overview

## Project Title

Personal Portfolio Website

## Description

This project was developed as part of Week 1 of The Developer Arena Web Development Internship Program.

The objective of this project is to create a simple and professional personal portfolio website using HTML5. The website demonstrates the use of semantic HTML elements, hyperlinks, images, forms, and navigation menus.

## Objectives

- Learn HTML5 document structure.
- Understand semantic HTML elements.
- Create a multi-section portfolio webpage.
- Add images and hyperlinks.
- Create a contact form.
- Implement internal navigation links.

## Features

- Header section
- Navigation menu
- About Me section
- Skills section
- Projects section
- Image gallery
- Contact form
- Footer section

## Technologies Used

- HTML5
- Visual Studio Code
- Git
- GitHub

## Learning Outcomes

Through this project, I learned:
- HTML document structure
- Semantic HTML tags
- Working with images
- Creating forms
- Internal page navigation
- Organizing project files
- Version control using Git and GitHub
