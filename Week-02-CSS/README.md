# Portfolio Website - Week 2 CSS Project

## Overview

This project is a responsive personal portfolio website developed using **HTML5** and **CSS3** as part of **The Developer Arena Internship Program - Week 2 CSS Assignment**.

The website showcases personal information, skills, projects, career goals, image gallery, contact form, and social media profiles while demonstrating modern CSS styling and responsive web design techniques.
