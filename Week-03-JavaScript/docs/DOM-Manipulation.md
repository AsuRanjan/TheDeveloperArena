# DOM Manipulation

The project uses JavaScript DOM APIs extensively.

## Elements Accessed

- Hero Title
- Greeting
- Navbar
- Contact Form
- Gallery Images
- Footer
- Buttons

Methods Used

- getElementById()
- querySelector()
- querySelectorAll()
- createElement()
- appendChild()
- insertBefore()
- insertAdjacentElement()

Events Used

- click
- dblclick
- mouseover
- mouseout
- keydown
- scroll
- input
- submit
- load
- DOMContentLoaded
