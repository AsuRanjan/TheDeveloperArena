# Week 03 - JavaScript Features

This document explains every JavaScript feature implemented in the Portfolio Website.

## Features

- Dynamic Greeting
- Live Digital Clock
- Dark Mode
- Form Validation
- Local Storage
- Session Storage
- Image Gallery
- Auto Slideshow
- Keyboard Navigation
- About Section Toggle
- Character Counter
- Live Preview
- Navigation Highlight
- Scroll Progress Bar
- Random Developer Quotes
- Profile Image Animation
- Scroll To Top
- Welcome Alert
- Footer Auto Year
- Liquid Glass Navigation

All features are implemented using Vanilla JavaScript without external libraries.
