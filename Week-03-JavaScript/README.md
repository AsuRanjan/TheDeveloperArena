Week 3: JavaScript - Making Websites Interactive

Theory Concepts:
- What is JavaScript? Understanding client-side scripting
- JavaScript basics: variables, data types, operators
- Functions and event handling
- DOM manipulation: selecting and modifying elements
- Event listeners: click, hover, submit, input
- Basic form validation with JavaScript

Hands-On Practice:
- Create external JavaScript file (script.js) and link it to HTML
- Write functions to show/hide content
- Create interactive image slider or gallery
- Build a to-do list with add/remove functionality
- Implement contact form validation
- Create dark/light mode toggle

Technical Requirements:
- Create script.js file
- Implement form validation with error messages
- Add at least 3 interactive features
- Use DOM manipulation to update page content
- Implement multiple event listeners
- Create reusable JavaScript functions

Interactive Features Implemented:
- Contact form validation
- Dark/Light mode toggle
- Interactive image gallery
- Dynamic greeting/message
- Smooth UI interactions
